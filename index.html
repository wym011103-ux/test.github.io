Production Diary – Week 5

Date: 13 February 2026

This week I worked on my GitHub Pages portfolio website. I focused on creating or improving the main sections (About, Projects, Skills, Contact) and setting up my CS Maths & Tools page.

I am most happy with learning basic knowledge through practice. For example: creating a basic website that only has basic functions such as play video and audio could help me connect the foundations all together. It's really easy to memorize and review the knowledge this way.

I found it difficult to learn some abstract concepts, such as different types of image licenses and how they work. It is not only hard to understand the logic behind them, but I am also frequently interrupted by many technical terms.

Next time I want to improve my ability to understand abstract problems. I believe that using AI tools or asking professionals, such as tutors, for help will make it easier to solve problems that involve technical terms or complex concepts.

Q: What did you actually build or change this week?

I learned a large amount of fundamental knowledge about loops in OOP and practiced using them, including the usage of for, do-while, and while loops. I also explored the differences in how loops are implemented in Java and Python. Through this process, I initially discovered that Java is very strict about data types and the use of braces. However, this strict structure helps prevent many errors and makes programs easier to maintain. In contrast, Python is very beginner-friendly and does not always require explicit data type declarations. While this makes it easier to get started, it can also lead to certain problems. For example, directly outputting a string and an integer together using the print function may result in an error. Moreover, such errors are sometimes not immediately obvious, which can make debugging more challenging.

Q: Which achievement are you most proud of so far on your website?

My website is very well-organized, simple, and easy to read.

Q: What did you find most challenging?

I am not very good at dealing with some logical problems, especially nested structures.

Q: What did you learn about web development or computer science maths from this work?

I believe this course has helped me fully develop my practical skills as well as my logical thinking ability.

Q: What is one concrete improvement you want to make next week?

I think AI is a very useful tool. I don’t let it do the work for me directly; instead, I use it to help explain concepts. It is very thoughtful in breaking down all the details in simple language. It can also help me expand on related topics and fill in the gaps in my understanding.
